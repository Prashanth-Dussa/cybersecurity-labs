# Acunetix Web Vulnerability Scan

## Objective
Identify vulnerabilities in a web application using Acunetix.

## Tool Used
Acunetix Web Vulnerability Scanner

## Target

Test Web Application (OWASP Juice Shop or local test site)

## Steps

1. Start Acunetix
2. Create a new scan
3. Enter target website URL
4. Start vulnerability scan
5. Review detected vulnerabilities

## Vulnerabilities Found

- Cross Site Scripting (XSS)
- SQL Injection Risk
- Weak Cookies

## Result

Acunetix identified several web vulnerabilities that could allow attackers to compromise the application.

## Learning Outcome

- Web application vulnerability testing
- Understanding OWASP vulnerabilities
- Analyzing automated scan reports
