# Wireshark Network Packet Analysis

## Objective
Analyze network traffic and identify protocols using Wireshark.

## Tool Used
Wireshark

## Steps

1. Start packet capture
2. Generate network traffic
3. Apply filters

Filters Used

http
dns
tcp

## Findings

Observed DNS queries and HTTP requests during the capture session.

## Learning Outcome

Understanding how network traffic flows and how attackers analyze packets.
