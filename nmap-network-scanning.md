# Nmap Network Scanning Lab

## Objective
To identify open ports and services running on a target machine.

## Tool Used
Nmap

## Command Used

nmap -sV 10.1.1.15

## Result

Open Ports Found:
21 - FTP
22 - SSH
23 - Telnet
80 - HTTP

## Learning Outcome

This lab helped me understand how attackers perform reconnaissance on networks to discover services and vulnerabilities.
